Vad kan man göra med CSS?

CSS kan ändra utseendet på en webbsida dramatiskt.

Du kan ändra färger och teckensnitt, layout och position, animera element med mera.

Byt stylesheet:

Starta projektet genom att

  1. Klona projektet från GitHub med git clone
  2. cd css-exercise/
  3. npm run start
  4. Du får en länk som börjar med "localhost". Öppna länken i en webbläsare.
  5. Skapa en egen CSS-fil i mappen styles/.
  6. Lägg till namnet på din fil i arrayen styles i script.js och ladda om sidan.
